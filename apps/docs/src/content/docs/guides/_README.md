# Development Journey Guides

Chronological documentation of how the Soon application was built.

## Purpose

- **Tutorial**: Follow along to understand how the project evolved
- **AI Context**: Provides context for Claude Code and other AI assistants
- **Decision Log**: Records architectural choices and their rationale

## Guide Structure

Guides are numbered sequentially (01-18+). Each follows a consistent format:

- **What**: What was accomplished
- **Why**: Rationale behind decisions
- **How**: Implementation details and commands
- **Context for AI**: Key patterns AI assistants should know

## Adding New Guides

1. Copy `_template.md` to `XX-your-feature.md` (next number)
2. Fill in all sections
3. Add to sidebar in `apps/docs/astro.config.mjs`
4. Update `00-overview.md` timeline
